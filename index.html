- Projects -

Guess the Word Game

An interactive game built with JavaScript that allows the user to make guesses to discover the hidden mystery word! Correctly guessed letters will display on the hidden word in real time. After all 8 chances are used, the user has the option to begin a new game, and the game restarts with a brand new randomly-fetched word. Play as many times as you like!

Live Page

Skills:
JavaScript, Responsive Design, Loops, Manipulating the DOM, CSS

Tools:
GitHub, Visual Studio Code

Name Tag App

An app that creates bulk name tags using functional components and ES6 modules. The user types in a name and submits it to create a new name tag featuring the entered name. The user can remove a name tag by clicking on the 'X' in the upper right corner of the name tag.

Live Page

Skills:
ES6, JSX, Managing Data Flow, React

Tools:
CodeSandbox

Sticky Notes App

An interactive React app that allows users to create, edit and delete virtual sticky notes. Users can also search for their to-do list items. The app also uses local storage to save the user's notes even when they close the browser window.

ES6, React, JSX, Manipulating the DOM, Managing Data Flow
